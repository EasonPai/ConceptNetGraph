It is built from nodes representing concepts, in the form of words or short phrases of natural language, and labeled relationships between them. These are the kinds of things computers need to know to search for information better, answer questions, and understand people's goals." by ConceptNet

About ConceptNetGraph

ConceptNetGraph is a data visualization app for ConceptNet. With ConceptNetGraph, you can browse concepts with visual graph as well as JSON data.

ConceptNetGraph is an experimental app build by Eason Pai and made with

  • Dart Client
  • Dart Server (Host on Heroku)
  • StageXL
  • and a lot of awesome Dart libraries

    • This app is still work in progress, more fun stuffs are coming up.
